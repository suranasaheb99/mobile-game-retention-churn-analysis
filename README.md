## Objective
Analyze player behavior to understand retention, churn, and engagement in a LiveOps-driven mobile game.

## Key Metrics
- DAU / MAU
- D1, D7, D30 Retention
- Churn Rate
- Session Frequency
- Cohort Analysis

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook

## Key Insights
- Strong D1 retention indicates good onboarding
- Sharp drop after Day 3 suggests early churn risk
- Newer cohorts show improved retention due to LiveOps effects

## Business Impact
Recommendations include onboarding rewards, daily missions, and time-bound events.
